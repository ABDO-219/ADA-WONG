# ADA-WONG

أداة `ADA-WING` لتشغيل أوامر نظام Linux Lite.

## الاستخدام

- `./ada-wing sysinfo`
- `./ada-wing update`
- `./ada-wing clean`
- `./ada-wing netcheck --host 8.8.8.8`
- `./ada-wing pkg-search firefox`
- `./ada-wing service status ssh`
- `./ada-wing user add testuser`
- `./ada-wing find log --path /var/log`
- `./ada-wing monitor`
- `./ada-wing disk`
- `./ada-wing shell`

## الواجهة الرسومية

لتشغيل الواجهة الرسومية بعد التثبيت:

- `ada-wing-gui`

أو من قائمة التطبيقات بعد تشغيل `./install-ada-wing.sh`.
